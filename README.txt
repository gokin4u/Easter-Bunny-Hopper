An interactive, physics-based bunny that rewards users with WooCommerce discount codes. Perfect for engagement and seasonal promotions.

== Description ==

Easter Bunny Hopper brings a delightful interactive experience to your WordPress site. It features a physics-driven bunny that users can click, drag, and interact with. 

As a reward for engagement, the bunny reveals a random WooCommerce coupon code after a certain amount of interaction (3 clicks). This is a fantastic way to boost conversion rates during holidays or special sales events.

== Features ==

*   **Interactive Physics**: Real-time bunny movement with gravity, drag, and collision effects.
*   **Engagement Rewards**: Automatically distribute WooCommerce discount codes to active users.
*   **Easy Configuration**: Select which coupons to rotate and customize bunny greetings directly from the admin dashboard.
*   **Elementor Ready**: Scoped CSS ensures the bunny works perfectly with page builders like Elementor without styling conflicts.
*   **Lightweight**: Uses SVG for sharp, scalable visuals that don't slow down your site.

== Installation ==

1. Upload the `easter-bunny-hopper` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > BunnyHopper** to select your coupons and customize greetings.

== Frequently Asked Questions ==

= Does it require WooCommerce? =
Yes, the coupon distribution feature requires WooCommerce to be installed and active.

= Can I customize the greetings? =
Absolutely! You can set custom greetings in the plugin settings.
